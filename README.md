![Screenshot](https://raw.github.com/marlesson/gosu_pong/master/screenshot.png)

# Setup

Dependencies:

* [RVM](https://rvm.io/)
* [bundler](http://bundler.io/)

```
git clone git://github.com/marlesson/gosu_pong.git
cd gosu_pong
bundle install
```

# Running

```
bundle exec ruby main.rb
```
